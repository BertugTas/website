# Logo 3D Export Notes

- Source file: `logo-3d-source.svg`
- Import into Blender as SVG.
- Convert curves to mesh, then use a small extrude value (`0.04` to `0.08`).
- Keep cyan nodes as separate materials for emissive accents.
- Export as `logo-mark.glb` for web use.

Target budgets:
- Mobile: under 35k triangles
- Desktop: under 80k triangles
